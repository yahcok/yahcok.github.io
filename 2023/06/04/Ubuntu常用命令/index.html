Ubuntu常用命令

调整缩放比例

  1. 利用sudo apt install gnome-tweak-tool命令安装tweak工具
  2. 在菜单搜索tweak工具
  3. 选择字体，调整缩放比例即可

修改火狐浏览器语言

  1. 输入命令sudo apt-get remove firefox卸载火狐浏览器
  2. 输入sudo apt-get install firefox安装火狐浏览器
  3. 输入sudo apt-get install firefox-locale-zh-hans 安装中文简体语言包

切换root用户

​ 第一次使用su root切换时会报su: Authentication failure（验证错误）

  1. 使用sudo passwd root给root账户设置密码，再重新输入
  2. 再使用su root命令切换root账户即可

切换普通用户

su 用户名

安装gcc

  1. 更新包列表sudo apt update
  2. 安装build-essential软件包sudo apt install build-essential软件包中包括gcc,g++,make
  3. 验证是否安装成功gcc -v

出现VMware tools not enough free…

  1. 将VMwareTools-10.3.10-13959562.tar.gz复制到其他路径，然后将其他路径的gz解压，这里可以看到右键是没有解压选项的，所以需要打开gz文件，将vmware-tools-distrib文件右键Extract到其他路径
  2. 将解压好的vmware-tools-distrib文件打开，右键根目录，open in Terminal，执行sudo ./vmware-install.pl命令，一路回车即可
